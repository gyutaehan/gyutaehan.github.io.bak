oh-my-zsh 설치 후 Insecure completion-dependent directories detected 에러 메시지가 나왔을 때 해결방법

에러 내용 oh-my-zsh 설치 후, 아래와 같은 경고 메시지가 나온다. Insecure completion-dependent directories detected ... 해결 방법 퍼미션 문제가 있는 디렉토리를 찾는다. $ compaudit # output /usr/local/share/zsh /usr/local/share/zsh/site-functions 문제가 있는 디렉토리를 퍼미션 설정을 해준다. $ chmod 755 /usr/local/share/zsh $ chmod 755 /usr/local/share/zsh/
